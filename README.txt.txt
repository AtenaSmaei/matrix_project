پروژه : پیاده سازی کتابخانه عملیات ماتریسی و برداری 

پروژه از سه بخش اصلی تشکیل شده است:
​فایل هدر (matrix_library.h): شامل  اسامی توابع مربوط به ماتریس 
​فایل (matrix_library.cpp): شامل منطق ریاضی و پیاده‌سازی جزئیات محاسبات ماتریسی است.
​فایل اصلی (demonstration.cpp): نقطه شروع برنامه (تابع main) که ورودی‌ها را از کاربر می‌گیرد و نتایج را نمایش می‌دهد.
​
​نحوه اجرای برنامه:
​ابتدا تمامی فایل‌های پروژه را در یک پوشه قرار دهید.  
​با استفاده از یک کامپایلر C++ (مانند g++)، فایل‌ها را کامپایل کنید: 

g++ *.cpp -o matrix_exe.exe

.\matrix_exe.exe

محاسبات برداری: عملیات ضرب خارجی صرفاً برای بردارهای ۳ بعدی (3D) پیاده‌سازی شده است

​مدیریت خطا: برنامه در صورت عدم تطابق ابعاد (در ضرب ماتریس‌ها و جمع و تفریق و معکوس و ...)
